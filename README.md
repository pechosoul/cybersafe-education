# CyberSafe Education

## Deskripsi

CyberSafe Education adalah website edukasi keamanan siber yang dibuat untuk meningkatkan kesadaran masyarakat mengenai pentingnya keamanan informasi dan perlindungan data pribadi di era digital.

Website ini dikembangkan sebagai tugas UAS mata kuliah Security Awareness and Digital Protection Product.

---

## Fitur Website

### 1. Home
Halaman utama yang menjelaskan tujuan dan manfaat website.

### 2. Artikel Cyber Security
Berisi materi edukasi mengenai:
- Phishing
- Password Security
- Two Factor Authentication (2FA)

### 3. Tips Keamanan Digital
Menyediakan tips praktis untuk menjaga keamanan akun dan data pribadi.

### 4. Simulasi Phishing
Memberikan contoh serangan phishing dan edukasi mengenai cara mengenalinya.

### 5. Kuis Keamanan Informasi
Kuis interaktif untuk menguji pemahaman pengguna mengenai cyber security.

## Cara Menjalankan Website

1. Download atau clone repository ini.
2. Buka folder project menggunakan Visual Studio Code.
3. Jalankan file `index.html` menggunakan browser atau extension Live Server.
4. Website siap digunakan.

---

## Tujuan Pembelajaran

Melalui website ini, pengguna dapat:

- Memahami ancaman siber yang umum terjadi.
- Mengenali ciri-ciri phishing.
- Mengetahui pentingnya password yang kuat.
- Memahami penggunaan Two Factor Authentication (2FA).
- Meningkatkan kesadaran terhadap keamanan informasi.
